# Samsung-CiberDeck

Transformar un smartphone Samsung GT-B5510L de 2011 en una terminal portatil para administracion de sistemas Linux.

## Objetivos
- Reutilizar hardware antiguo.
- Aprender administracion de sistemas Linux
- Experimentar con SSH y acceso remoto
- Construir una plataforma para futuros proyectos NAS

## Desafios 
- Android 2.3.66 obsoleto
- Compatibilidad limitada de aplicaciones
- Acceso root
- Configuracion SSH

## Soluciones implementadas
- Root del dispositivo
- Limpieza de aplicaciones innecesarias
- Instalacion de herramientas de terminal
- Conexion con servidores Debian y FreeBSD

## Estado actual

✅ Acceso SSH funcional
✅ Optimizacion del sistema

- Android 2.3.66
- Linux
