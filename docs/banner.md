# 🖥️ Banner CyberDeck

## Objetivo

Mostrar información personalizada al iniciar una sesión SSH.

---

## Archivo utilizado

```bash
nano ~/.bashrc
```

---

## Información mostrada

* Usuario
* Host
* Fecha
* Hora
* Dirección IP
* Cliente conectado

---

## Futuras mejoras

* Detección automática Samsung
* Cliente desconocido
* Estado NAS
* Estado ESP32
* Información sensores

---

## Resultado

Al iniciar SSH:

```text
====================================
 Samsung CyberDeck
 Debian Remote Terminal
====================================
```
