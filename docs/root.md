
# 🔓 Root Samsung GT-B5510L

## Objetivo

Obtener acceso root sobre Android 2.3.6 Gingerbread para permitir modificaciones avanzadas del sistema, eliminación de aplicaciones de fábrica y personalización del dispositivo.

---

## Dispositivo

| Característica | Valor             |
| -------------- | ----------------- |
| Modelo         | Samsung GT-B5510L |
| Android        | 2.3.6 Gingerbread |
| CPU            | Qualcomm MSM7227  |
| Teclado        | QWERTY físico     |
| Estado         | Root funcional    |

---

## Herramientas utilizadas

* Universal GB Root v20
* Recovery Stock Samsung
* SuperSU

---

## Procedimiento

### 1. Copiar ZIP a la microSD

```text
universal_gb_root_v20.zip
```

### 2. Ingresar al Recovery

```text
Vol+ + Home + Power
```

### 3. Instalar ZIP

```text
Apply update from sdcard
```

### 4. Reiniciar

```text
Reboot System Now
```
